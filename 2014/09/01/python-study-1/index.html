PYTHON基础（一）

python

前言

在学习廖雪峰的PYTHON教程的过程中，备忘一些需要注意的知识点，特别感谢作者能够提供这么好的教程！此外推荐其关于GIT的教程，写的也相当不错。

字符串和编码

1、普通的字符串在Python内部都是ASCII编码，Python提供ord()和chr()函数。ord()将字母转换成数字，chr()将数字转换成字母。

2、添加了对Unicode的支持，例如：u'你好'，否则直接print '你好'，Python会报错。

3、字符串'xxx'既可以看作ASCII编码也可以看作是UTF-8编码，u'xxx'只能是Unicode编码。

list

学习过java，对list应该不会陌生，但是和java里的list肯定不一样（废话）。

1、list是一种有序的集合，可以随时添加和删除元素。e.g. colors = ['red','black','yellow']就是一个list。

2、list里面的元素的类型可以不同，还可以包含另一个list，e.g.language = ['English','German',['Putonghua','Cantonese'],'Korean']，如果要获的元素'Putonghua'，可以使用language[2][0]。

3、支持两种索引方式：如果从第一个元素开始，那么索引就为0；如果从最后一个元素开始，那么索引就为-1。因此colors[0]和colors[-3]都表示red这个元素，但是都要确保索引不要越界，否则报错。

4、len()函数可以获得list元素的个数，空list的长度为0。

5、常用操作

  • 追加元素到末尾
  • 删除末尾元素
  • 将元素插入到指定位置
  • 删除指定位置的元素
  • 将某个元素替换成别的元素

tuple

tuple和list都是Python内置的有序集合，只是一个可变，一个不可变，而tuple是“不可变”的，这里的“不可变”准确的是说tuple初始化后指向的元素不可变，并不表示tuple的内容不可变。（和java中的final类有点类似。）

1、定义tuple的同时需要进行初始化，tuple中的元素也就确定了。

2、空tuple：()，是一对圆括号，而空list：[]，是一对方括号。

3、定义只有一个元素的tuple，在元素后面必须加一个,，否则()会作为数学公式中的括号进行计算。

dict

全称dictionary，dict使用键-值(key-value)存储，写法类似javascript中的字面量对象，e.g. d = {‘zhangsan’: 18,’lisi’: 19, ‘wanger’: 20}

1、一个key只能对应一个value，多次对一个key进行赋值，后面的值会覆盖前面的值。

2、dict是根据key的值计算value的存储位置，作为key的对象必须是不可变的。因此list就不能作为key，那么tuple能吗？

3、常用操作

  • 使用in判断key是否存在。
  • 使用get方法，如果key不存在默认返回None，或者自己指定返回的值；如果key存在返回value的值。
  • 删除一个key使用pop(key)方法，对应的value也会删除。

set

set是一组key的集合，不存储value，并且key不重复。 创建一个set，需要一个list作为输入集合。

1、通过add(key)和removw(key)实现元素的添加和删除，可以重复添加，但没有什么意义；remove(key)的时候，如果key不存在，则会报错。

2、set可以被看成数学意义上的集合，因此两个set可以做数学意义上的交集、并集等操作；

3、和dict一样，不可以放入可变对象，因为无法判断两个可变对象是否相等，也就无法保证set内部元素不会重复。

小结

其实上述的list、dict以及set和java中的list、map以及set很像，学过java的理解这几个概念应该会很快。
